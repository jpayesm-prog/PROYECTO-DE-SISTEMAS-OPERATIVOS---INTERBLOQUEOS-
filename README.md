# PROYECTO-DE-SISTEMAS-OPERATIVOS---INTERBLOQUEOS-
Este proyecto es un simulador visual de sistemas concurrentes. Permite el estudio práctico del interbloqueo (deadlock) mediante la detección por grafos de espera y la recuperación por selección de víctimas. Combina teoría de S.O. con una GUI intuitiva para visualizar en tiempo real cómo los procesos se bloquean y generan ciclos.
